# Stage 03: Chinh phục Cloud Kingdom – Key takeaways

## Push nhánh lên GitHub

Ở Cloud Kingdom người ta dùng **git branch** rất nhiều. Hai câu lệnh bổ sung:

### Push một nhánh lên GitHub

```bash
git push origin <tên_nhánh>
```

**Ví dụ:**
```bash
git push origin main
git push origin feat/giai-cuu
```

### Nhớ

- Push **tất cả** các nhánh đã tạo lên GitHub để mentor chấm bài được.
